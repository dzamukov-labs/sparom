# ⚠️ ВАЖНО: НЕ КОНВЕРТИРОВАТЬ В ASTRO

Эта папка `/lp` содержит **отдельные лендинги** на чистом HTML/CSS/JS.

## Правила:

1. **НЕ трогать** эту папку при миграции на Astro
2. **НЕ конвертировать** файлы в .astro формат
3. **НЕ переносить** в src/pages/
4. Лендинги работают **независимо** от основного сайта

## Структура:

```
/lp/
  └── bani-01/          ← Отдельный лендинг
      ├── index.html    ← Обычный HTML
      ├── css/
      └── js/
```

## Почему?

Лендинги могут иметь другую структуру, A/B тесты, разные версии.
Они живут отдельно и не должны зависеть от основного сайта.
